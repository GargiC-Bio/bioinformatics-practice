# StrokeGenomics: Computational Tools for Inherited Risk Analysis

## Project Overview
This repository contains bioinformatics tools developed to identify genetic proneness for chronic stroke.This script calculates the GC content of a DNA sequence to help determine its thermal stability. By analyzing genomic data, specifically focusing on GC content and sequence stability, I aim to map regions associated with neuro-vascular health.
## Features
- *GC Content Analysis*: Automatically calculates the Guanine-Cytosine percentage of DNA sequences to identify high-stability genomic regions. 
- *Python-Based Processing*: Analizes efficient string manipulation to handle genetic data. 
- *Future Integration*: Planned support for analyzing larger datasets from the 'Python for Genomic Data Science' curriculum.
